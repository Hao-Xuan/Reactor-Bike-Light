# Mobile Application

This directory contains representative portions of the Reactor mobile application developed in React Native.

Included files demonstrate:

* BLE communication utilities
* Redux state management
* Device synchronization logic
* Virtual device controls and indicators
* User-interface component organization

Navigation, styling, assets, build configuration, and other framework-specific files have been omitted for brevity. The selected files are intended to showcase application architecture and communication between the mobile app and embedded device.
